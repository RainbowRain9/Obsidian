## 3.2 学生-课程数据库

### 3.2.1 数据库模式说明

- **模式名称**：S-T（Student-Course）
- **包含表**：
    1. `Student`（学生表）
    2. `Course`（课程表）
    3. `SC`（选课表）
- **用途**：用于教学演示SQL操作的基础场景

---

### 3.2.2 表结构定义

### 1. Student（学生表）

| 字段名   | 数据类型     | 约束          | 说明       |
| ----- | -------- | ----------- | -------- |
| Sno   | CHAR(9)  | PRIMARY KEY | 学号（唯一标识） |
| Sname | CHAR(20) | UNIQUE      | 学生姓名     |
| Ssex  | CHAR(2)  | -           | 性别       |
| Sage  | SMALLINT | -           | 年龄       |
| Sdept | CHAR(20) | -           | 所在院系     |

**示例数据**：

```SQL
INSERT INTO Student VALUES
('201215121', '李勇', '男', 20, 'CS'),
('201215122', '刘晨', '女', 19, 'CS'),
('201215123', '王敏', '女', 18, 'MA'),
('201215125', '张立', '男', 19, 'IS');
```

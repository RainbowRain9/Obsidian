好，我们来小试牛刀一下。比如说我在优化《二哥的 Java 进阶之路》中“第一行 Java 代码时”时，需要生成 hello world 的样板代码，那我只需要按下 enter 键换一行。

## 02、代码问答

我个人比较常用的就是代码问答，毕竟上下文的交互能力对于我们程序员来说更加重要，因为开发中难免要碰到一些比较复杂的业务场景，或者遇到一些我们不熟悉的编程语言时，能够帮助我们快速进行学习。
